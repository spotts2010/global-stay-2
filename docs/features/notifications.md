# Notification Center Features

The Notification Center is a centralized hub for all user communications.

## Features

- **Unread Highlighting**: Distinct visual styling for unread items.
- **Rich Filtering**: Filter by "All", "Read", or "Unread".
- **Search**: Keyword search across titles and summaries.
- **Importance Flagging**: Users can flag notifications to exempt them from auto-archiving.
- **Type-Specific Icons**: Visual cues for Payments, Holidays, Requests, and System Alerts.
