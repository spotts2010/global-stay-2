# User Account & Dashboard

The Account section allows users to manage their presence on the platform.

## Personal Details

- Editable profile with avatar upload and timezone management.
- Dynamic preference settings for Currency and Language.

## Travel Documents

- Secure section for users to store and manage their driver's license, passport, and travel insurance details.
- Standardized fields for quick data entry during future bookings.

## Favorites System

- Users can save and manage a list of their favorite properties.
- Favorites are persisted in the browser's local storage for instant retrieval.

## Dashboard Widgets

- Customizable overview for Guests (upcoming trips), Hosts (performance metrics), and Admins (system health).
