# Search System

The search system is the primary discovery mechanism for Global Stay 2.0.

## Implementation

- **Homepage Search**: A multi-input form for Location, Dates, and Guests.
- **Location Search**: Integrated with Google Places Autocomplete for high accuracy.
- **Date Picking**: Uses `react-day-picker` with custom styling for holiday range selection.
- **Results Page**:
  - **Card View**: Displays property highlights, live pricing, and ratings.
  - **Map View**: Advanced marker placement with interactive InfoWindows.
- **Filtering**: Real-time filtering by price range, property type, and minimum rating.
