# Listings Feature

Listings are organized into Properties (Accommodations) and Units.

## Accommodation Details

- Detailed amenities and accessibility feature lists.
- Interactive map showing the exact location.
- Photo gallery with full-screen viewer.
- Real-time "Lowest Price" calculation from available units.

## Booking Workflow

- Users browse properties, view available units, and select a specific room or bed.
- Selection redirects to a confirmation flow with pricing breakdowns.

## Status Management

- **Draft**: Work in progress, not visible to guests.
- **Published**: Live and bookable. Requires at least one image and valid location data.
- **Archived**: Hidden from the site but preserved for historical records.
