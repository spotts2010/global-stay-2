# Admin & Host Tools

A separate section for administrators to manage the global ecosystem.

## Listing Management

- Comprehensive table view with bulk status updates and duplication tools.
- Step-by-step editing for Property About, Photos, Amenities, and Policies.

## Unit Configuration

- Multi-unit support for complex properties (e.g., Hotels or Hostels).
- Granular pricing, bed configurations, and bathroom counts.

## System Administration

- Master list management for Amenities and Accessibility Features.
- Site-wide settings for Hero Image libraries.
- Legal page content management with a Rich Text Editor (TipTap).
