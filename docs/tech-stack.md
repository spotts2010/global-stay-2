# Technology Stack

## Core

- **Next.js 15**: Framework for React applications.
- **TypeScript**: Static typing for improved developer productivity and code quality.
- **Tailwind CSS**: Utility-first CSS framework for styling.

## Backend & Data

- **Firebase Client SDK**: Direct database interaction from the client.
- **Firebase Admin SDK**: Secure server-side actions.
- **Firestore**: Real-time NoSQL database.
- **Firebase Storage**: Cloud file storage.

## UI & Components

- **ShadCN UI**: Reusable component primitives.
- **Radix UI**: Underlying accessible headless primitives.
- **Lucide React / React Icons**: Standardized iconography.
- **Embla Carousel**: For high-performance carousels.

## AI & Tooling

- **Genkit**: Framework for AI features.
- **Zod**: Schema validation for data and AI inputs.
- **Jest & Testing Library**: Unit and integration testing.
- **Playwright**: End-to-end testing.
