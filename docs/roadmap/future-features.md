# Future Roadmap

## High Priority

- **Unit Availability Indicator**: Live count of published units on the booking card.
- **Dynamic Collections**: Associate listings with collections based on Firestore tags.
- **Secure Document Hosting**: Encrypted storage for passports and licenses.

## Regional & Localization

- **Dynamic Language/Currency Selector**: Full actionable indicator in the header.
- **Translations API**: Integration for global language support.
- **Currency Conversion API**: Real-time rates instead of fixed tables.

## Host & Admin Enhancements

- **Seasonal Rates**: Date-range based pricing for units.
- **Address-to-Timezone Mapping**: Automated timezone detection.
- **Bulk Imports**: Support for adding multiple POIs or Listings via CSV.

## Security & Verification

- **Two-Factor Authentication (2FA)**: Via SMS and Email.
- **Avatar Moderation**: AI-assisted review of user-uploaded images.
- **Verified Host Badge**: Verification workflow for hosts.
